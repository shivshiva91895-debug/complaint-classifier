# AI Customer Complaint Classifier

This project is a Machine Learning based web application that classifies customer complaints automatically.

## Technologies Used
- Python
- Flask
- Machine Learning
- Scikit-learn
- Chart.js

## Features
- Classifies complaints into:
  - Delivery Issues
  - Refund Issues
  - Payment Issues
  - Product Issues
- Stores complaint history
- Shows complaint statistics
- Displays complaint charts

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open in browser:

http://127.0.0.1:5000
